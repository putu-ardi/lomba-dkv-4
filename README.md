# Media Pembelajaran Interaktif - DKV Grafika v 1.0 (Juni 2026)

Media Pembelajaran Interaktif untuk Materi I: Wawasan Dunia Kerja Bidang DKV dan Grafika (SMK/Vokasi Kelas X).
Berbasis Standalone HTML + JavaScript tanpa server.

## Cara Penggunaan
1. **Akses Online**: Buka URL GitHub Pages yang tersedia.
2. **Offline**: Aplikasi otomatis menyimpan cache saat pertama kali dibuka. Untuk penggunaan offline sepenuhnya, unduh repositori ini sebagai ZIP, ekstrak, dan buka `index.html` di browser.
3. **Progress**: Seluruh progress belajar dan skor kuis disimpan otomatis di browser siswa masing-masing.

## Struktur Modul
* Modul 1A - Peta Profesi DKV & Grafika
* Modul 1B - Teknologi, Tren & Keberlanjutan
* Modul 1C - Bahasa Visual & Design Thinking
* Kuis Akhir Komprehensif
